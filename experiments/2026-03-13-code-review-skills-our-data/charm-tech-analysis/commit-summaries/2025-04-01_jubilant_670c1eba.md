## Summary

Breaking API change: deploy() parameter renamed from singular 'resource' to 'resources' for a dict param

## Commit Message

This is based on feedback from Jon that the singular is just weird here
as it's always a dict, never a single item. I think this is fair, and we
don't have to be quite so pedantic about the 1:1 match to the CLI arg.
