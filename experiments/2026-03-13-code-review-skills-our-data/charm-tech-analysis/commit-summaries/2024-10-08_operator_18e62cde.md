## Summary

remove enum | str unions in private typeddicts

## Commit Message

These TypedDicts represent values returned from pebble, which will only contain
simple serializable values like strings and ints and dicts of strings and ints.
We can simplify the types by removing the use of enums in these TypedDicts.

Note that removing the use of such unions throughout this module as planned in
issue #1287 not possible, due to the need to maintain backwards compatibility
-- union types like this are exposed on the public attributes of objects like
Check, CheckInfo, FileInfo, and so on.
