## Summary

## Commit Message

Removes an outdated (or perhaps always wrong) check in Harness that
prevents accessing the remote app databag during relation-broken events.
Juju does allow this, so Harness should have the same behaviour.

This check is specific to Harness and there's no equivalent check in
Scenario.

Fixes #888
