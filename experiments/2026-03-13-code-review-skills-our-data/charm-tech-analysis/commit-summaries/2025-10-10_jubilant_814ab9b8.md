## Summary

Revealed secrets crashed with KeyError on Juju < 3.6 which lacks the 'checksum' field

## Commit Message

A secret object doesn't have a `checksum` field in older versions of
Juju (3.0 ~ 3.5).

This change uses an empty string for a secret in such cases.
