## Commit Message

Small fixes for the `hookcmds` package:

* When flattening action parameters, recurse into any mapping, not just
mutable mappings (this leaves the `model` version unchanged, but that's
more specifically expecting a dict)
* Correct the return type of `action-get`, which can be any action
parameter type, which means essentially `Any`
* Correct the docstring of `relation-get`'s `unit` argument

These were reported by @carlcsaposs-canonical , thanks!
