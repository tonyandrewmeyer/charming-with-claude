## Summary

## Commit Message

Fixes
https://github.com/canonical/pebble/actions/runs/9392436751/job/25867410242

Recent [CI
runs](https://github.com/canonical/pebble/actions/runs/9392436751/job/25867410242)
started failing as Snapcraft is no longer considering the legacy
location for Launchpad credentials,
"$XDG_DATA_DIR/snapcraft/provider/launchpad/credentials".

This PR changes the snap.yml such that it now uses
"$XDG_DATA_DIR/snapcraft/launchpad-credentials" instead.

Reference:
https://git.launchpad.net/snapcraft/commit/?id=83b67923d11200bc8dd8e75ec46df2644d70659d
