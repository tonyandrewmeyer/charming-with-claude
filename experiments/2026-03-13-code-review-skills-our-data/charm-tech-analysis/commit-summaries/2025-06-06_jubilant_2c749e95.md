## Summary

juju offer command was incorrectly passing --model flag which Juju CLI rejects

## Commit Message

When implementing the `Juju.offer` method, I had overlooked that that
command doesn't take a `--model` argument, and Juju complains when you
do. Remove it.

This command doesn't have any integration tests, but update the unit
tests with a negative test of this.

Fixes #147.
