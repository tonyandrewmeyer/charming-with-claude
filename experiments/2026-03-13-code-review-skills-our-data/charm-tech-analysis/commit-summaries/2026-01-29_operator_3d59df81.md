## Summary

make testing.checkinfo level arg type match pebble.checkinfo.level

## Commit Message

This PR adds a custom `__init__` method to `testing.CheckInfo` to allow
the `level` argument to accept a `str` (which is then converted to a
`pebble.CheckInfo` before setting the `testing.CheckInfo.level`
attribute). This allows the `level` argument to be a
`pebble.CheckInfo.level` (e.g. pulled from a `pebble.Layer`) without the
type checker complaining.

+@pytest.mark.parametrize(('level'), ['', 'alive', 'ready'])

+    check_info = CheckInfo(name='Fëanor', level=level)
+    assert check_info.level is ops.pebble.CheckLevel(level)
