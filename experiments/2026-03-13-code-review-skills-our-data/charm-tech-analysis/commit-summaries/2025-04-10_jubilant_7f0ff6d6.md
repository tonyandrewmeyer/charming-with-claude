## Summary

Every juju status poll in wait() was logged, creating excessive noise; also fixed blank-line glitches in diffs

## Commit Message

This removes the logging of every `cli: juju status ...` call while
polling status in `Juju.wait` sequences.

Also removes certain "glitches" where it'd show a status change when
only `.since` timestamp lines changed, and would result in blank lines
like this:

```
INFO wait: status changed:

INFO ... more logs ...
```

Fix #86
