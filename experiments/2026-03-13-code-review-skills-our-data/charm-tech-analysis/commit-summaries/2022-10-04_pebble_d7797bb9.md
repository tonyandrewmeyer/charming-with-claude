## Summary

Add missing ErrDarwin error for Darwin platform compilation compatibility

## Commit Message

When copying code from snapd to pebble we missed ErrDarwin so that
pebble can be compiled on Darwin.

See
https://github.com/snapcore/snapd/blob/61b66aaa1beb861f308ea7dd75c2f0c042bf29a3/osutil/osutil_darwin.go
for reference.
