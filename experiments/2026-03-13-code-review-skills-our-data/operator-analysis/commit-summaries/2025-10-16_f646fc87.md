fix!: change JujuContext.machine_id from int to str (#2108)

The `JUJU_MACHINE_ID` context variable sometimes looks like an integer,
but is actually typed as a string, and can be a non-integer string in
some cases.

The field in `JujuContext` is changed, as well as the corresponding
field in `testing.Context`.

Fixes #2107
